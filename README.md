# Smart Revision Tracker

A Python-based application that helps students track revision performance across subjects and topics.

## Features
- Add revision data (subject, topic, score)
- Identify weakest topics
- Calculate average scores per subject
- Save and load data using JSON

## Technologies Used
- Python
- JSON

## Purpose
This project was built to improve revision efficiency by analysing performance data and identifying weak areas.

## Future Improvements
- Add graphs for progress tracking
- Build a graphical user interface (GUI)
